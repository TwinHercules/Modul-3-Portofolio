MY PROJECT

WEBSITE DONASI SMAMS 1 GRESIK

Website ini dikembangkan untuk memfasilitasi penggalangan donasi di SMAM 1 Gresik. Platform ini menampilkan informasi seputar kegiatan sosial, memudahkan proses donasi, dan menghubungkan donatur dengan berbagai program kemanusiaan yang diadakan sekolah.

PROJECT AKHIR ALPRO RUMAH SAKIT

Merupakan proyek akhir mata kuliah Algoritma dan Pemrograman (ALPRO) berupa simulasi sistem informasi rumah sakit. Project ini fokus pada pengelolaan data pasien, dokter, dan pelayanan medis dengan menggunakan bahasa pemrograman untuk mendukung efisiensi administrasi rumah sakit.

PROJECT PORTOFOLIO

Website portofolio pribadi yang dirancang untuk menampilkan profil, riwayat pendidikan, proyek-proyek, serta kontak. Website ini berfungsi sebagai media memperkenalkan diri secara profesional kepada dunia luar, terutama untuk keperluan akademik maupun karier.
